Usage example for [ifcpp](https://github.com/ExternPointer/ifcpp)

All third-party included as git submodules.

Clone:
```
git clone https://github.com/ExternPointer/ifcpp-example.git
cd ifcpp-example
git submodule init
git submodule update
```
